# JustGoogle

[![PWA Version (not Flutter)](https://img.shields.io/badge/Website-live-green)](https://cmutnik.github.io/JustGoogle/) <-- PWA Version (not Flutter)

A search engine with no AI components. Back to the basics.

To run this locally, you need a `.env` file with the following variables:

```bash
GOOGLE_API_KEY=<your google api key here>
GOOGLE_SEARCH_ENGINE_ID=<your google search engine id here>
```

then launch the app with:

```bash
flutter run
```
